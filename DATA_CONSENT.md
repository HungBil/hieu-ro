# Data Consent

RAG may ingest only approved writing samples that are:

- consented by the user
- anonymized
- marked non-sensitive
- approved by reviewer/admin workflow

Do not ingest raw phone numbers, emails, IDs, health details, bank details, or names unless already anonymized.

Community posts, user samples, saved phrases, and learning data must never be seeded as fake runtime data. Empty databases should show empty states.
